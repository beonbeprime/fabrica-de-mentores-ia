## 🛠️ Como Usar

Voce precisa ter o [Claude Code](https://docs.anthropic.com/en/docs/claude-code) instalado na sua maquina.

1. Clone este repositorio:
```bash
git clone https://github.com/beonbeprime/fabrica-de-mentores-ia.git
```

2. Navegue ate a pasta do agente que quer usar:
```bash
cd fabrica-de-mentores-ia/arquiteto-de-mentoria
```

3. Inicie o Claude Code:
```bash
claude
```

4. O agente vai carregar automaticamente as instrucoes da pasta e esta pronto para trabalhar.

Cada pasta tem um `CLAUDE.md` com as instrucoes especificas do agente. O Claude Code le esse arquivo e ja sabe exatamente o que fazer.
